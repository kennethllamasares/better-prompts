# Changelog

All notable changes to the "Better Prompts" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2025-01-06

### Changed
- Added marketplace icon

## [0.1.0] - 2025-01-06

### Added
- Initial release
- Sidebar UI for prompt generation
- AI enhancement with Ollama (free, local)
- AI enhancement with Anthropic Claude API
- AI enhancement with OpenAI GPT API
- Rule-based enhancement (offline fallback)
- Context management with quick add buttons
  - Add current file
  - Add selected code
  - Browse and add files
- Context chips with remove functionality
- Intent detection for rule-based mode
- Multiple intent templates (fix, add, change, explain, test, review, improve, document)
- Status bar showing current AI mode
- Manual refresh button for status
- Settings for API configuration
- Context menu integration
- Keyboard shortcuts
- Auto-detection of available AI providers
